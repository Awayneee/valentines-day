# Sevgililer Gunu Mini Site

Bu proje mobil uyumlu tek sayfalik bir surpriz sitesidir.

## Yerel acma
- `index.html` dosyasini tarayicida ac.

## Ucretsiz yayinlama (onerilen: GitHub Pages)
1. GitHub'da yeni bir repo olustur.
2. Bu dosyalari repoya yukle.
3. Repo ayarlarinda `Pages` bolumune gir.
4. `Deploy from a branch` sec, `main` + `/root` sec.
5. Site su adreste acilir: `https://kullaniciadiniz.github.io/repo-adi/`

## Alternatif ucretsiz yayinlama
- Netlify: Drag-and-drop ile yayinlayip ucretsiz alt alan adi alabilirsin.
- Cloudflare Pages: GitHub baglayip tek tikla yayinlama.

## Kisisellestirme
- Sorular: `index.html`
- Dogru cevaplar: `script.js` icindeki `answers`
- Mesaj ve imza: `index.html` icindeki `#surprise`
